== Description ==
create quizzes with easy navigations. view result in a pie chart.
view solutions and leaderboard.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload qnotsquiz.zip file in plugins folder.
2. Activate and create questions. Add questions to quizzes.
3. Find the shortcode for each quiz at quiz dashboard
4. Paste the shortcode anywhere in yours posts or pages
5. Thats it. Your users can start the quiz now.
